Web Technologies

HTML

HTML (HypterText Markup Language) is the fundamental building block of the Web and defines the content and structure of a web page. Below are some common page elements that HTML is used for:

  • Headings
  • Paragraphs
  • Images
  • Links
  • Lists

CSS

CSS (Cascading Style Sheet) is used to describe a web page's apperance and presentation. Below are some common uses of CSS:

  • Fonts
  • Colours
  • Layout
  • Backgrounds

Learn more about CSS with w3schools CSS Tutorial
